# KYフィードバック品質テストフィールド

## 概要

KY（危険予知）フィードバックの品質向上機能を評価するための独立したテストツールです。

**特徴**:

- 3モデル（gpt-4o-mini / gpt-5.2 / gpt-5-mini）の比較
- 5つのプロンプトバリエーション
- 応答のブレ確認（各3run）
- ユーザー評価入力＆統計グラフ

## 使い方

### 1. ブラウザで開く

```bash
# ローカルサーバーを起動（CORS対策）
cd apps/v2-modern/experiments/feedback-quality-test
npx serve .
# http://localhost:3000 を開く
```

または、VSCodeのLive Server拡張機能で `index.html` を直接開く。

### 2. テスト実行

1. **テストケース** を選択
2. **プロンプト** を選択
3. **Run** を選択（ブレ確認用）
4. **比較実行** をクリック

### 3. 評価入力

- 各モデルの応答に対して5段階評価
- メモを記録
- **評価を保存** でローカルストレージに保存
- **JSONエクスポート** でファイル出力

## ファイル構成

```
feedback-quality-test/
├── index.html         # テストUI
├── test-cases.json    # 10件のテストケース
├── prompts.json       # 5つのプロンプト定義
├── mock-responses.json # モック応答データ
└── README.md          # このファイル
```

## プロンプト一覧

| ID | 名前 | 特徴 |
|---|---|---|
| v1_current | 現行 | 既存プロンプト |
| v2_specific | 具体性強化 | 数値・事例を要求 |
| v3_field_adapted | 現場適応 | 現場制約を考慮 |
| v4_structured | 構造化 | 段階的分析 |
| v5_motivational | モチベーション重視 | 前向き表現 |

## モデル特性（シミュレーション）

| モデル | 特徴 |
|---|---|
| gpt-4o-mini | 標準的、コンパクト |
| gpt-5.2 | 高精度、詳細、論理的 |
| gpt-5-mini | バランス、実用的 |

## 注意事項

- このツールは **Antigravityがシミュレート** した応答を使用
- 実際のAPIは呼び出していません
- 評価データはブラウザのローカルストレージに保存されます
