PROJECT VOICE-KY-ASSISTANT // v2.0-modern

AIファシリ型 危険予知（KY）システム ─ 技術概要とビジネスメリット

2026.03

開発動機：KY活動の形骸化

  • 1. 一人KYの限界：他者の視点がなく「ヨシ」で済ませる思考停止。潜在リスクが見落とされる。
  • 2. 手書き記入の面倒さ：細かな欄に長文を書く手間。チェック項目の増加もあり、億劫さから形式的な記入に。
  • 3. 二度書きの無駄：元請用と自社用で同じ内容を繰り返し記録する非生産的な運用。

解決策：中核機能

■ 音声対話による「ファシリテーション」

AIが深掘り質問を行いマンネリを防止。過去事例から「新たな視点」をサジェストし気付きを促進。

■ ハンズフリー入力で「負荷ゼロ化」

AIとの自然な会話だけでKY活動が完結。手袋のまま音声だけで記録を作成。

■ 監査対応PDFの即時「自動生成」

対話から構造化データを抽出し、捺印可能なKYシートPDFを即座に出力。二度書きを根本排除。

対応環境・利用条件

  • ●動作環境：iPhone / Android / PC のブラウザ（PWA対応）
  • ●インストール：不要。URLアクセスで即利用開始
  • ●通信環境：AI対話時にネット接続が必要。入力内容はオフラインでも保持
  • ●セキュリティ：APIキー等は中継サーバー（BFF）で隠蔽し、セキュアに通信可能
  • ●データ保管：端末ローカルに保存（外部保管なし）

システム処理フロー

1. 音声入力（PWA / ブラウザ）

Web Speech APIでリアルタイムにテキスト化。テキスト入力にも対応。

▼

2. BFF / API Gateway（CF Workers）

認証・レート制限・会話履歴管理。APIキーをクライアントから隔離。

▼

3. AI判定・構造化抽出

Gemini / OpenAI が返答と同時にJSON形式でKYデータを構造化抽出。

▼

4. PDF出力・ローカル保存

React-PDFで帳票を即時生成。IndexedDBに30日間オフライン保管。
