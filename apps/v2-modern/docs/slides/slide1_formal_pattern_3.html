STRATEGIC ROADMAP

Voice KY Assistant 中長期ロードマップとビジョン

作成・組織展開から「全社ダッシュボード連携・データ資産化」までの4フェーズ計画

MISSION 「二度書きの無駄をなくし、現場の知恵をつなぐ」

フェーズの進行に伴い、対象ユーザーを「個人」から「組織・チーム」へ拡大。 現場作業者の負担ゼロから始まり、最終段階では蓄積されたデータが組織横断的な安全指針（ダッシュボード分析・複数人参加のチームKY）へと進化し、現場の労働災害をシステム的に予防する資産へと飛躍します。

P1

MVP / 実証実験

利用規模: 1人 (開発・PoC)

  • 音声対話の実現可能性検証
  • 最低限のPDF出力機能実装
  • 完全ローカル保存による基盤構築
P2
CURRENT

ソロKYの安定化と定着

利用規模: 1〜10人規模

  • マンネリ化を防ぐ「新たな視点」の提供
  • 音声対話とレスポンスの安定稼働化
  • 高品質レコード(PDF)の自動生成
P3

情報の統合と組織展開

利用規模: 10〜100人規模

  • 他社指定KY用紙のOCR取り込み変換
  • 現場写真・画像認識による危険箇所提示
  • CSV等へのサーバー定期的エクスポート抽出
P4

分析活用とチームKY

利用規模: 100〜200人〜

  • ダッシュボードによる現場横断の分析
  • チームKY（1台端末で複数人の発言を統合）
  • 集合知による高度な予知インサイト生成

各フェーズ進行に伴うKPI目標値達成に向けて：

段階的な機能リリースにより利用者の負担感を払拭しつつ、最終目標値へと堅実に推移させます。

建築KY取り込み率目標 (P3)

70% 以上

15分 →8分

ROI・コスト維持 (恒久)

5円 / 回 以下
