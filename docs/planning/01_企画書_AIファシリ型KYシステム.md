## 1. エグゼクティブサマリー

### 背景

建設現場における危険予知（KY）活動は、労働災害防止の要であるにも関わらず、**形骸化**が進んでいる。作業員は「書くのが面倒」「考えるのが苦手」という二重の負担を抱えており、結果として毎回同じ内容を記入する、あるいは適当に済ませるという状況が発生している。

### 提案

**音声AIがKY活動をファシリテート**することで、「書く」負担をゼロにし、「考える」をAIがサポートするシステムを開発する。

### 選定アプローチ: AIファシリテート型（案B）

AIが一方的に答えを提示するのではなく、**対話を通じて作業員自身が考える**ことを促す。抽象的な回答には深掘り質問をすることで、具体的で質の高いKYを実現する。

### 将来展望

MVPで効果を確認後、**AIコーチ型（案C）**へ段階的に拡張。

- 危険候補の提案機能
- 国のヒヤリハットデータ活用
- スコアリング・分析機能

## 2. プロジェクト概要

### 2.1 ビジョン
>
> **「話すだけで、質の高いKYが完了する」**

### 2.2 成功基準

| 指標 | 目標 |
|------|------|
| **KYの質** | 具体的な危険・対策が挙げられるようになる |
| **利用継続** | 作業員が「使いたい」と思う |

### 2.3 ターゲットユーザー

| 区分 | 対象 | 特徴 |
|------|------|------|
| 主要ユーザー | 建設現場の作業員 | 一人でKY実施、スマホ操作やや不慣れ |
| 端末 | iPhone SE（会社支給） | 小画面、音声操作が重要 |

### 2.4 対象作業

- **足場設置作業** に特化（初期スコープ）
- 1日2回（朝・昼）のKY実施を想定

## 3. 課題と解決策

### 3.1 現状の課題

| 課題 | 詳細 |
|------|------|
| 書くのが面倒 | 手が汚れている、紙に記入するのが煩わしい |
| 考えるのが苦手 | 何が危険か、どう対策するか思いつかない |
| 形骸化 | 毎回同じ内容、やらされ感 |

### 3.2 本システムによる解決

| 課題 | 解決策 |
|------|--------|
| 書くのが面倒 | **音声入力**で完結、手ぶらでOK |
| 考えるのが苦手 | **AIが深掘り質問**で考えを引き出す |
| 形骸化 | **天候・作業に応じた動的質問**でマンネリ防止 |

## 4. システム概要

### 4.1 基本フロー

```
[アプリ起動]
     │
     ▼
[作業選択] ←── ボタンタップ（足場設置）
     │
     ▼
┌─────────────────────────────────────┐
│        AI対話セッション             │
│                                     │
│  AI: 「今日の作業で、               │
│       何が危ないと思いますか？」    │
│           ↓                         │
│  作業員: 「落ちるかも」             │
│           ↓                         │
│  AI: 「どの作業の時に？             │
│       もう少し具体的に」            │
│           ↓                         │
│  作業員: 「板を置くとき」           │
│           ↓                         │
│  AI: 「対策は？」                   │
│           ↓                         │
│  作業員: 「安全帯かける」           │
│           ↓                         │
│  AI: 「今日の合言葉は？」           │
│           ↓                         │
│  作業員: 「安全帯ヨシ！」           │
│                                     │
└─────────────────────────────────────┘
     │
     ▼
[確認画面] ←── 内容を確認、修正可能
     │
     ▼
[完了] ←── PDF自動生成、保存
```

### 4.2 AIの役割

| 役割 | 詳細 |
|------|------|
| **質問生成** | 作業種別・天候に応じた質問を生成 |
| **深掘り** | 抽象的な回答には具体化を促す |
| **整理** | 発言を構造化してKY記録にまとめる |

## 5. MVP機能一覧

### 5.1 必須機能（MVP）

| 機能 | 説明 | 優先度 |
|------|------|--------|
| 🎙️ 音声KY実施 | AIとの対話でKYを完了 | 必須 |

| 💬 AI深掘り質問 | 抽象的回答に追加質問 | 必須 |
| 📄 PDF生成 | KY記録をPDF出力 | 必須 |
| 💾 記録保存 | クラウドに保存 | 必須 |

### 5.2 後から追加（将来）

| 機能 | 説明 | フェーズ |
|------|------|----------|
