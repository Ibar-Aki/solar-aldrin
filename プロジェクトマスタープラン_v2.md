# AIファシリ型KYシステム（Voice KY Assistant）Project Master Plan v2

**「話すだけで、質の高い危険予知が完了する」**
**「二度書きの無駄をなくし、現場の知恵をつなぐ」**

本ドキュメントは、AIファシリ型KYシステム（Voice KY Assistant）の**ビジョン、対象範囲、ロードマップ、投資対効果、成功指標、リスク対策、実行体制**を統合した最上位の企画書である。

---

## 1. エグゼクティブサマリー

### 1.1 背景と課題
建設現場における危険予知（KY）活動は労働災害防止の要である。しかし、以下の課題によって**形骸化**が進んでいる。

1. **記入負担の肥大化**: 「書くのが面倒」「考えるのが苦手」という二重の負担が現場で常態化している。
2. **二重入力のムダ**: 元請け提出用と自社保管用のKY用紙が異なる場合、同じ内容を二度書く必要がある。
3. **知見の分断**: KY記録が紙で埋もれ、過去のヒヤリハットや他者の気付きが共有されない。

### 1.2 解決方針
音声AIがKY活動をファシリテートし、以下を実現する。

* **入力の省力化**: 音声入力と画像取り込みで「書く」負担をゼロに近づける。
* **思考の深化**: 対話によって潜在的な危険を深掘りする。
* **成果物の自動化**: 監査に耐えるKY記録（PDF）を自動生成し、再利用可能なデータ資産にする。

## 2. 目的・成功指標（KPI）

### 2.1 目的
* KY活動の質と量を向上させ、労働災害の防止に寄与する。
* 事務作業の負担を削減し、現場の生産性を向上させる。

### 2.2 成功指標（KPI）
以下を**Phase 2完了時の最低到達目標**とする。

| 指標 | 目標値 | 測定方法 |
| :--- | :--- | :--- |
| KY作成時間の中央値 | 10分 → 3分以内 | アプリ計測（開始〜PDF生成） |
| 1人あたり月間KY作成数 | 20%向上 | 利用ログ集計 |
| 音声入力成功率 | 95%以上 | 音声認識の完了率 |
| PDF生成成功率 | 99%以上 | 生成処理の成功ログ |

---

## 3. 対象範囲（スコープ）

### 3.1 スコープ内
* Phase 2における**ソロKYの音声対話**
* **PDF自動生成**とプレビュー
* **シンプルなデータ保存**（ユーザー端末のLocal Storage）

### 3.2 スコープ外（Phase 2では実施しない）
* 他社KY用紙のOCR取り込み
* 現場写真診断
* 複数人の会話を統合するチームKY
* 組織横断のダッシュボード分析

## 5. 投資対効果（ROI）

### 5.1 コスト構造
* サーバー費用: Cloudflare Workers無料枠内で運用可能
* AI利用料: GPT-4o-miniを利用し1回あたり1.5円以下を目標

### 5.2 期待効果
* 記入時間の削減（70%以上）
* 記述品質の向上（具体的な危険箇所記載の増加）
* 監査対応工数の削減

## 6. リスクと対策

| リスク | 原因 | 影響 | 対策 |
| :--- | :--- | :--- | :--- |
| 音声認識精度低下 | 現場の騒音、方言 | KY内容の欠落 | ノイズ低減、再入力UIの準備 |
| OCR精度不足 | 手書き文字のばらつき | 二度書き解消に失敗 | OCR結果の確認・修正機能 |
| ネットワーク不安定 | 山間部・地下 | データ消失の恐れ | Local Storageの利用と再送機能 |
| セキュリティ事故 | データ漏えい | 信頼失墜 | 暗号化保存、アクセス制御 |
