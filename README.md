# Voice KY Assistant

> 建設現場のKY（危険予知）活動をAI音声でサポートするシステム

## 📋 概要

作業員がスマートフォンに話しかけるだけで、AIが対話を通じてKY活動をファシリテート。
「書くのが面倒」「考えるのが苦手」という課題を解決し、質の高いKYを実現します。

## 🎯 特徴

- **音声入力** - 手ぶらでKYが完了
- **AI深掘り** - 抽象的な回答を具体化
- **PDF自動生成** - 記録が自動で残る
